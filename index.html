GitHub

Git is a fast, open source, distributed version control system that is quickly replacing subversion in open source and corporate programming communities. Very briefly, version control is a way to keep a backup of changing files, to store a history of those changes, and most importantly to allow many people in a collaboration to make changes to the same files concurrently.

JavaScript

JavaScript (JS) is an interpreted computer programming language. It was originally implemented as part of web browsers so that client-side scripts may interact with the user, control the browser, communicate asynchronously and alter the document content that is displayed.
